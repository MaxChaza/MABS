Reference Column Filter

This node allows columns to be filtered from the first table using the second table as reference table. Depending on the dialog setting, either the columns from the reference table are included or excluded in the output table.

Dialog Options

In-/Exclude columns from reference table
Includes or excludes all columns from the reference table into the resulting table from the first input.
Ensure compatibility of column types
Ensures that the matching columns don't only have the same name but also the same type. Columns are only included or excluded if the column type of the first table is a super-type of the column type from the second table. If not selected, only the column names need to match.

Ports

Input Ports
0 Table from which columns are to be included or excluded
1 Table with the columns used as references
Output Ports
0 Table with filtered columns
