Iterate List of Files

Description:

Iteratively executes the contained flow on a list of files. The list of files needs to be defined by the input table, whereby each row represents one individual file location.

Contained nodes:

Variable Based File Reader (deprecated):
ASCII file reader from variable locations
TableRow To Variable Loop Start:
Iterates over an input data table, whereby each row defines on iteration with variable settings taken from the values in that row
Loop End:
Node at the end of a loop
