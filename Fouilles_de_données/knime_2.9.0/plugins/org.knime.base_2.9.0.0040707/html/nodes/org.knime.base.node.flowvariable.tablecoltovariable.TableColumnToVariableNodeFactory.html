Table Column to Variable

Converts the values from a table column to flow variables with the row IDs as their variable name and the values in the selected column as the variable values.

Dialog Options

Column name
Name of the column for the values.
Skip missing values
When checked, rows with a missing value in the selected column will be skipped. When unchecked, the node execution will fail if a row contains a missing value in the selected column.

Ports

Input Ports
0 A table
Output Ports
0 New flow variables with names from the row IDs.
