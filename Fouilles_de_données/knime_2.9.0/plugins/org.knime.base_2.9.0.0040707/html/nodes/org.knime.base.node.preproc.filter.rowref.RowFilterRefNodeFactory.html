Reference Row Filter

This node allows rows to be filtered from the first table using the second table as reference. Depending on the dialog setting, the rows from the reference table are either included or excluded in the output table.
During the test for in-/exclusion the values of the selected columns of both tables are compared.

Dialog Options

Data table column
The column from the table to be filtered that should be used for comparison.
Reference table column
The column from the filter table that should be used for comparison.
In-/Exclude rows from reference table
Includes or excludes all rows from the reference table in the resulting table from the first input.

Ports

Input Ports
0 Table from which rows are to be included or excluded
1 Table rows used as reference filter
Output Ports
0 Table with filtered rows
